# NayePankh Foundation 

A minimalist full-stack web application developed to manage volunteer submissions and handle administrative operations securely.



## 🛠️ Key Technologies
- **Frontend:** React, Vite, Axios, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Mongoose ODM)
- **Security:** JSON Web Tokens (JWT), bcyptsjs, cookie-parser, cors
